# ⚡ Landing Page - Pikachu

## 📌 Sobre o Projeto

Este projeto consiste no desenvolvimento de uma **landing page temática inspirada no personagem Pikachu**, do universo Pokémon.

O projeto foi desenvolvido **em conjunto com o professor Leonid**, com o objetivo de aplicar na prática conceitos de estruturação e estilização de páginas web.

---

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3

---

## 🎯 Objetivo

O principal objetivo deste projeto foi:

* Praticar a estruturação de páginas utilizando HTML
* Trabalhar estilização e layout com CSS
* Reproduzir um design moderno e atrativo
* Aplicar conceitos aprendidos em aula

---

## ⚙️ Funcionalidades

* Layout visual temático inspirado no Pikachu
* Organização de elementos como título, descrição e botão
* Design focado em estética e simplicidade
